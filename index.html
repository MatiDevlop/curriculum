Proyectos

  • CRUD con MySQL y Python

    Es una aplicación diseñada para la empresa de Tiendas Industriales Asociadas Tia S.A. cuyo propósito principal es gestionar el estado y los movimientos de los productos que tienen todos los almacenes. Las tecnologías aplicadas fueron una base de datos MySQL y para la vista del usuario se implemento una interfaz en Python.

  • Plataforma de comercialización de vehículos

    Se desarrollo una plataforma web para publicar, buscar y filtrar vehículos en venta. Las tecnologías aplicadas al proyecto fueron el lenguaje Java para las reglas de negocio, JavaFX para la interfaz gráfica GUI; y como concepto principal se emplearon diversas estructuras de datos.

    Ver repositorio

  • Sistema de toma de decisiones

    Es un juego donde el programa realiza una serie de preguntas al usuario en base a las caracteristicas de los animal es que el mismo conoce. El usuario solo puede responder sí o no y el programa toma las decisiones. Las tecnologías usadas fueron el el lenguaje Java para la lógica de programación, JavaFX para la interfaz GUI; y como concepto principal se usaron los arboles de decicion como estructura de datos.
